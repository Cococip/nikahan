# The Archive: Wedding Documentary

A high-end, A24-inspired minimalist digital wedding invitation/documentary website designed specifically for a Videographer and Reporter couple.

## 🎬 Concept
"The Archive" replaces the traditional "wedding invitation" with a cinematic, highly-polished digital exhibition. 
- **Aesthetic**: Dark Mode Minimalist Brutalism.
- **Typography**: Editorial Serif (`Playfair Display`, `Noto Serif KR`) paired with technical Sans-Serif (`Inter`, `Noto Sans KR`).
- **Features**: Smooth Lenis scrolling, GSAP parallax animations, looping video backgrounds, and a hidden YouTube iframe background score.

## 🚀 Features
- **The Gate (Cover)**: Full-screen video loop overlay that slides up like a theatrical curtain using GSAP.
- **Seamless Marquee**: Pure CSS infinite scrolling typography.
- **Editorial Grid**: Asymmetric, high-fashion layout with GSAP parallax triggers.
- **RSVP Form**: Clean, bottom-bordered inputs with micro-interactions.

## 🛠 Tech Stack
- HTML5 / CSS3 (Vanilla)
- Vanilla JavaScript
- [GSAP (GreenSock)](https://greensock.com/) - For timeline & scroll animations
- [Lenis Scroll](https://lenis.studiofreight.com/) - For butter-smooth scroll inertia
- [Lucide Icons](https://lucide.dev/) - For crisp UI icons

## 📂 File Structure
```text
.
├── index.html       # Main HTML structure
├── css/
│   └── style.css    # All styling & responsive rules
├── js/
│   └── script.js    # GSAP logic, Lenis setup, and UI interactions
└── assets/          # Contains images (1.jpeg - 10.jpeg, cowo.png, cewe.jpeg) and video (vid.mp4)
```

## 💻 How to Run
Simply open `index.html` in any modern web browser or serve via a local development server (e.g., VS Code Live Server).
